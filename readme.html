Pretty How Type: A CSS Typography Kit

Version v0.5 // 20130517

Pretty How Type is a readability-focused CSS typography framework in sans-serif with strong vertical rhythm. It was built with mobile and single-column layouts in mind and is comparable to popular apps and plugins to increase article readability on mobile devices.

Pretty How Type's heading typeface is Montserrat, a free display font developed by Julieta Ulanovsky. There are known issues with non-local fonts on some browser/OS combinations (like Chrome/Windows). Where this is a concern, I recommend falling back onto Verdana (the body typeface).

Pretty How Type uses a font-size of 16px and a line height of 24px, but it resizes well for accessibility.

Feedback and suggestions are always welcome: bartjhelms@gmail.com.

Usage

The current release includes

  • Consistent formatting for ul, ol, dl, blockquote, and code.
  • Auto-shrinking for embedded media within article tags.
  • Support for figure, figcaption.
  • Forms following the accessibility guidelines here.
  • A set of formatting classes such as alignleft, alignright, aligncenter, .nodecoration. .pullquote

Download

Feel free to @import the live version into your CSS, but be forewarned that I will not announce minor updates for the time being. Version controlled updates will appear on GitHub.

If you desire strict baseline vertical spacing and embedded media are a major component of the design, I recommend using baseline.js to resize elements.
